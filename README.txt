Description
============
Rayan store is a simple product store.


Purpose
=======
Demonstrate Simple Authentication without ASP.NET Identity Core.

Details
========
- It has a frontend and backend.
- For backend, just add/edit/remove products
- Backend also is accessed through login feature
- Front end just to display products.

Plan
====
- Create entities (Product, User)
- Create Admin panel Product (Create, Edit, Delete)
- Create login/logout
- Create frontend
